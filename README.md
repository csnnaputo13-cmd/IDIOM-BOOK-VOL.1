# 📘 IDIOM·BOOK·VOL.1

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-All%20Rights%20Reserved-red)
![Platform](https://img.shields.io/badge/platform-Web%20%7C%20Mobile-green)

A comprehensive interactive English idiom learning and testing application designed for CSE, IELTS, TOEFL, and ESL learners.

---

## 🌟 Features

### 📖 Review Mode
- **50 Idiom Cards** - Browse through all 50 idioms with beautiful images
- **🔊 Text-to-Speech** - Listen to idiom meanings, examples, and usage with voice selection
- **🔍 Zoom & Pan** - Tap or click images to zoom in and drag to explore details
- **📄 Pagination** - Navigate easily with Prev/Next buttons or Go-To page
- **🎯 Visual Learning** - Each idiom comes with a visual representation

### 📝 Test Mode
- **50 Multiple-Choice Questions** - Test your knowledge of all 50 idioms
- **✅ Submit Answer** - Get detailed step-by-step analysis for each answer
- **📊 Step-by-Step Analysis** - Includes meaning, word definition, context analysis, conclusion, and references
- **🎯 Performance Rating** - Get rated based on your score
- **🏆 Pass/Fail Status** - Passing score: 40/50 (80%)
- **📋 Detailed Results** - See which questions you got wrong with correct answers

### 🎨 Design
- **Dark Blue & Orange Theme** - Easy on the eyes with high contrast
- **💜 Purple Accents** - Professional and modern look
- **📱 Fully Responsive** - Works on desktop, tablet, and mobile
- **3D Tile Effects** - Modern button design with tactile feedback

## 🚀 How to Use

### Welcome Screen
1. **Enter your name** in the input field
2. **Choose your mode:**
   - 📖 **Review First** - Browse all idioms before testing
   - 📝 **Take Test** - Jump straight to the exam
3. Click **Start** or press **Enter**

### Review Mode
- Navigate with **Prev/Next** buttons or use **←/→** keyboard shortcuts
- **Zoom in/out** using the ➕/➖ buttons or pinch on mobile
- **Listen to audio** by clicking the 🔊 button
- **Change voice** using the voice selector dropdown
- Click **Back to Welcome** to return to the main menu

### Test Mode
1. Select your answer from the 4 options
2. Click **Submit Answer** to see analysis
3. Navigate with **Prev/Next** or **←/→** keyboard shortcuts
4. On Question #50, click **📊 Submit Exam** to finish
5. View your **Congratulations Popup** with score and rating
6. Click **🏆 View Results** for detailed breakdown
